Presso · Operations Wireframes
Screens 7 (Rider drop-at-facility) and 14 (Facility dispatch) are marked NEW — these are the missing handoff screens in the current codebase. All facility screens fail today because /api/facility/* endpoints don't exist on the backend. Status labels here (At Facility / Washing / Ironing / Ready) match the Flutter app; the API enum uses InProcess / ReadyForDelivery — pick one vocabulary before building.
